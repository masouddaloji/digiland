
# Progress Project (Backend)

This is the backend part of the progress project (e-commerce)


## Tech Stack

Node, Express, Bcrypt, Fastest Validator, JWT, multer, and much more


## Features

- admin routes
- article routes
- auth routes
- basket routes
- favorate (wishlist) routes
- order routes
- product routes
- review routes
- upload routes
- user routes


## Installation

Install my-project with npm

```bash
  cd my-project
  npm install
  npm run dev
```
